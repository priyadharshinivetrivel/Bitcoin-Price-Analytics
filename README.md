## Bitcoin Price Analytics and Forecasting

## Project Overview

This project analyzes historical Bitcoin market data and predicts future Bitcoin closing prices using Machine Learning techniques.

## Objectives

* Perform Exploratory Data Analysis (EDA)
* Analyze Bitcoin price trends
* Engineer predictive features
* Train and compare SVR and XGBoost models
* Forecast future Bitcoin prices
* Visualize insights using charts and graphs

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* XGBoost

## Dataset

Historical Bitcoin market data from 2020 to 2025.

## Machine Learning Models

1. Support Vector Regression (SVR)
2. XGBoost Regressor

## Evaluation Metrics

* Mean Absolute Error (MAE)
* Root Mean Squared Error (RMSE)
* R² Score

## Project Structure

data/ - Dataset

src/ - Source code

models/ - Trained models

visualizations/ - Graphs and charts

README.md - Project documentation

requirements.txt - Python dependencies
